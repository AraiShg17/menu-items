# CSS Cursor プロジェクトガイドライン

## 🎯 プロジェクトの目的

- Next.js を使用したモダンなWebアプリケーションの開発
- CSS Modules によるスタイル管理
- レスポンシブデザインの実装

## 📋 開発ルール

### 1. **フレームワーク**: Next.js 14+

- App Router を使用してください
- TypeScript で型安全性を確保してください

### 2. **スタイリング**: CSS Modules

- コンポーネント単位でCSSファイルを作成してください
- グローバルスタイルは `globals.css` に記述してください

### 3. **デザイン**: モダンなUI/UX

- oklab色空間のグラデーションを使用してください
- レスポンシブデザインを実装してください
- アクセシビリティを考慮してください

## 🏗️ プロジェクト構造

```
src/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # ルートレイアウト
│   ├── page.tsx           # メインページ
│   ├── globals.css        # グローバルスタイル
│   └── detail/
│       └── page.tsx       # 詳細ページ
├── components/            # 再利用可能なコンポーネント
│   └── Button/            # コンポーネント毎にフォルダにまとめる
│       ├── Button.tsx     # ボタンの実装
│       └── Button.module.css # ボタンのスタイル
```

## 🎨 デザインシステム

### カラーパレット

- **プライマリ**: oklabグラデーション（青→紫）
- **セカンダリ**: oklabグラデーション（オレンジ→ピンク）
- **背景**: oklabグラデーション（グレー系）

### タイポグラフィ

- **フォント**: Geist Sans（メイン）、Geist Mono（コード）
- **サイズ**: レスポンシブ対応

## 📱 レスポンシブデザイン

- モバイルファーストのアプローチ
- ブレークポイント: 768px, 1024px
- フレキシブルなレイアウト

## 🚀 開発環境

- Node.js 18+
- Next.js 14+
- TypeScript
- ESLint
